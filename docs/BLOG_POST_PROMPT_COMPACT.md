# Blog Post Creation Prompt (Compact Version)

Use this compact prompt to quickly generate SEO-optimized blog posts for AI Tales.

## Quick Prompt

```
You are an expert SEO content writer creating a comprehensive blog post for AI Tales.

TOPIC: [Insert topic]
PRIMARY KEYWORD: [Insert keyword]
TARGET AUDIENCE: [Insert audience, e.g., "parents of toddlers"]
AUTHOR: "AI Tales Team" OR "[Expert Name, Title]"

Create a 2,500-4,500 word blog post with:

**Frontmatter:**
- Title: 50-60 chars, include primary keyword, add "(2025 Guide)" if relevant
- Description: 150-160 chars, question or benefit statement
- Date: YYYY-MM-DD
- Category: Parenting Tips | Reading & Education | Education Science | Child Development | Sleep & Development | Story Ideas
- Keywords: 6-8 relevant keywords (primary + long-tail + question-based)
- Image: /blog/[slug].jpg

**Structure:**
1. Hook opening (quote/scenario/question)
2. Problem statement + promise
3. H2 sections (5-8): Understanding, Science/Research, Strategies, Mistakes, Age-Specific, FAQ, Conclusion
4. CTA at end (italicized, mentions AI Tales)

**Content Requirements:**
- Research-backed with 2024-2025 citations
- Actionable strategies with examples
- Age-specific guidance (Toddlers 1-3, Preschoolers 3-5, School-age 6-12)
- Empathetic, solution-focused tone
- Use bold, lists, blockquotes, tables
- Primary keyword in title, first paragraph, 2-3 H2 headings
- Include statistics, realistic examples (Emma, Kai, Jake, Maya)

**Format:**
- MDX file format
- Proper heading hierarchy (H1 → H2 → H3)
- Horizontal rule (---) before CTA
- Italicized CTA: "*Ready to [benefit]? Try AI Tales free and [specific value].*"

Generate the complete blog post now.
```
